## 项目介绍：
随着城市化进程加快，居民对就业和住房信息的需求日益增长。58同城作为国内领先的生活服务平台，汇聚了大量的招聘和房源数据。如何高效采集、清洗、分析这些数据，并为城市就业、房价趋势、投资决策等提供数据支持，成为数据分析和智能决策的重要课题。
本项目以58同城为数据源，结合爬虫技术、数据清洗、机器学习和可视化分析，旨在构建一套完整的城市就业与房价数据分析体系，为相关决策和研究提供有力的数据支撑。

本项目分为四个主要阶段：
1.	数据采集
利用Python爬虫自动化采集58同城各城市的招聘和房源信息，并存储到本地数据库。
2.	数据清洗与预处理
将原始数据上传至HDFS分布式文件系统，导入hive数据库。对原始数据进行清洗、去重、标准化处理，提取关键字段（如职位、薪资、房价、面积等），并构建机器学习特征表。
3.	数据分析与建模
通过Spark等大数据工具，对城市就业、房价、福利、投资性价比等进行统计分析和预测建模，生成多维度汇总数据。
4.	数据导出与可视化
将分析结果以中文列名CSV格式导出，便于在FineBI等可视化平台进行展示和深度分析，支持城市对比、趋势洞察和投资决策。
项目最终实现了从数据采集到分析、可视化的全流程自动化，提升了数据处理效率和分析深度，为城市就业与房价研究提供了高质量的数据基础。
